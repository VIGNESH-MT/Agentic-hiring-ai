# Beta-Version-AI-Resume-Screening-Skill-Search-Match-
An end-to-end system that can:  Analyze an open resume-screening dataset (EDA + cleaning)  Train an ML model to predict a hiring outcome  Build an agent that extracts skills from a real resume PDF and matches them to a skills catalog (HF + LangChain) with a UI  Implement the same workflow via an MCP server + MCP client agent
